# AccountType

The `seller` and `vendor` account types are associated with Sponsored Ads APIs. The `agency` account type is associated with DSP and Data Provider APIs.

## Enum

* `VENDOR` (value: `'vendor'`)

* `SELLER` (value: `'seller'`)

* `AGENCY` (value: `'agency'`)
