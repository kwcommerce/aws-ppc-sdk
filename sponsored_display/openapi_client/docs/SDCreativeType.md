# SDCreativeType

The type of the associated creative. If the field is empty or null, a default value of IMAGE will be used. Only supports one type (VIDEO or IMAGE) at a time.

## Enum

* `IMAGE` (value: `'IMAGE'`)

* `VIDEO` (value: `'VIDEO'`)
